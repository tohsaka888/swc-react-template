# swc-react-template

## introduction

use `swc` and `webpack5` to build the template for your react application.

## usage

clone this repository

### download dependencies

```bash
pnpm install
```

### run `dev` server

```bash
pnpm start
```
 
### build

```bash
pnpm build
```

The compiled files are in the `/dist` directory

### serve

```bash
pnpm serve
```
